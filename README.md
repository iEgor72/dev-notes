# Developer Notes

Personal knowledge base with short notes, examples and cheat sheets for software development.

The project contains compact documentation pages with syntax, explanations and practical examples.  
It is designed as a quick reference while learning and building projects.

## Live Demo

https://iegor72.github.io/dev-notes/

## Topics

Currently available notes include:

- HTML
- CSS
- Layout
- Git
- JavaScript
- TypeScript

More topics will be added over time.

## Features

- clean documentation layout
- table of contents navigation
- active section highlighting
- scroll progress indicator
- sticky navigation
- back-to-top button
- responsive design
