Что такое TypeScript

TypeScript — это надстройка над JavaScript, которая добавляет статическую типизацию.

TypeScript помогает находить ошибки ещё во время разработки и делает код более понятным и безопасным. 

let name: string = "Alex";
    let age: number = 25;

TypeScript компилируется в обычный JavaScript.

Базовые типы

Тип Пример
string "Hello"
number 42
boolean true / false
any любой тип
unknown неизвестный тип
void нет возвращаемого значения

Типизация переменных

let username: string = "Alex";
    let age: number = 30;
    let isAdmin: boolean = false;

TypeScript может автоматически определить тип:

let username = "Alex";

Типизация функций

function sum(a: number, b: number): number {
      return a + b;
    }
const greet = (name: string): string => {
      return "Hello " + name;
    }

Интерфейсы

interface User {
      name: string
      age: number
      isAdmin: boolean
    }
const user: User = {
      name: "Alex",
      age: 25,
      isAdmin: false
    }

Интерфейсы описывают структуру объектов.

Type aliases

type ID = string | number
type User = {
      name: string
      age: number
    }

Массивы

const numbers: number[] = [1,2,3]

    const users: User[] = []

Generics

function identity<T>(value: T): T {
      return value
    }
identity<number>(5)
    identity<string>("hello")

Generics позволяют писать универсальный код.

Классы

class User {
      name: string

      constructor(name: string) {
        this.name = name
      }
    }

Полезные utility типы

Тип Что делает
Partial<T> делает поля необязательными
Readonly<T> делает объект только для чтения
Pick<T> выбирает нужные поля
Omit<T> убирает поля

Что важно запомнить

  • TypeScript — это JavaScript + типы
  • типы помогают находить ошибки заранее
  • интерфейсы описывают структуру объектов
  • generics позволяют писать универсальный код
  • TypeScript компилируется в JavaScript
