Как подключить CSS

CSS можно подключить тремя основными способами: через отдельный файл, внутри тега <style> или прямо в атрибуте style.

Способы подключения CSS
Способ Пример Когда использовать
Внешний файл <link rel="stylesheet" href="style.css" /> Основной и самый удобный способ
Внутри страницы <style> p { color: red; } </style> Для маленьких демо или тестов
Inline-стиль <p style="color: red;">...</p> Редко, точечно, не для основной стилизации 
<link rel="stylesheet" href="style.css" />

В реальных проектах чаще всего используют внешний CSS-файл.

Синтаксис CSS

p {
  color: red;
  font-size: 18px;
}

CSS-правило состоит из селектора и блока объявлений. Внутри блока пишутся свойства и их значения.

Из чего состоит CSS-правило
Часть Что означает
p Селектор: выбирает, к каким элементам применить стиль
color Свойство: что именно меняем
red Значение: каким будет новое состояние свойства

После каждого свойства со значением ставится двоеточие, а в конце объявления — точка с запятой.

CSS-селекторы

p { }

.card { }

#header { }

.menu a { }

button:hover { }
  • p — селектор по тегу
  • .card — селектор по классу
  • #header — селектор по id
  • .menu a — селектор потомка
  • button:hover — стиль при наведении
Популярные типы селекторов
Селектор Что выбирает Пример
* Все элементы * { box-sizing: border-box; }
.class Элементы с классом .card { padding: 20px; }
#id Элемент с конкретным id #header { background: #111; }
a:hover Состояние элемента a:hover { color: tomato; }
.card p Элементы внутри другого элемента .card p { margin: 0; }

В реальной практике классы используют чаще, чем селекторы по тегу или id, потому что они удобнее и гибче.

Основные CSS-свойства

Часто используемые CSS-свойства
Свойство Что делает
color Цвет текста
background Фон элемента
font-size Размер текста
font-family Шрифт
margin Внешний отступ
padding Внутренний отступ
border Рамка элемента
border-radius Скругление углов
box-shadow Тень блока
width / height Ширина и высота элемента
display Определяет способ отображения элемента
max-width Ограничивает максимальную ширину
overflow Управляет тем, что выходит за границы блока

Box Model

.card {
  width: 300px;
  padding: 20px;
  border: 1px solid #ccc;
  margin: 20px;
}

Каждый HTML-элемент можно представить как прямоугольник, который состоит из: content, padding, border и margin.

  • content — содержимое элемента
  • padding — внутренний отступ
  • border — рамка
  • margin — внешний отступ
* {
  box-sizing: border-box;
}

Свойство box-sizing: border-box делает расчёт размеров удобнее: padding и border входят внутрь указанной ширины и высоты элемента.

Без border-box внутренние отступы увеличивают итоговый размер блока. С ним размеры становятся более предсказуемыми.

Margin и Padding

Разница между margin и padding
Свойство Что делает
margin Создаёт пространство снаружи элемента
padding Создаёт пространство внутри элемента 
.box {
  margin: 20px;
  padding: 20px;
}

Простое правило: margin — снаружи, padding — внутри.

Display

Популярные значения свойства display
Значение Описание
block Элемент занимает всю доступную строку
inline Элемент идёт в строку и обычно не принимает width/height
inline-block Идёт в строку, но можно задавать размеры
flex Включает Flexbox для удобного выравнивания элементов
grid Включает Grid для построения сеток
none Полностью скрывает элемент 
.menu {
  display: flex;
  gap: 12px;
}

Flexbox

Flexbox часто используют для выравнивания элементов в строку или колонку, а также для распределения пространства между ними. 

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 16px;
}
Базовые свойства Flexbox
Свойство Что делает
justify-content Выравнивает элементы по главной оси
align-items Выравнивает элементы по поперечной оси
gap Задаёт расстояние между элементами
flex-direction Меняет направление: строка или колонка

Для простого горизонтального или вертикального выравнивания Flexbox — один из самых удобных инструментов.

Overflow и z-index 

.card {
  overflow: hidden;
}

.modal {
  position: fixed;
  z-index: 100;
}
Полезные свойства для слоёв и переполнения
Свойство Что делает
overflow: hidden Скрывает то, что выходит за границы блока
overflow: auto Добавляет прокрутку при необходимости
z-index Определяет, какой элемент будет выше других

z-index обычно работает у элементов с position, отличным от static.

Единицы измерения

Частые единицы в CSS
Единица Что означает Когда использовать
px Пиксели Точные размеры: рамки, отступы, тени
% Проценты от родителя Гибкая ширина и размеры
em Относительно текущего размера шрифта Локальное масштабирование
rem Относительно размера шрифта корневого элемента Размеры текста и отступов по всей системе
vw / vh Относительно окна браузера Крупные адаптивные блоки и секции

Для типографики и отступов часто удобно использовать rem, а не только px.

Специфичность

Если к одному и тому же элементу подходят несколько правил, браузер решает, какое из них важнее. Это и называется специфичностью. 

p {
  color: blue;
}

.card p {
  color: green;
}

#intro {
  color: red;
}

Обычно приоритет растёт примерно так: селектор по тегу → класс → id.

Старайся не строить стили на множестве id и слишком длинных селекторах. Проще поддерживать стили, основанные на классах.

Каскад и порядок правил

CSS расшифровывается как Cascading Style Sheets. Это значит, что важен не только селектор, но и порядок, в котором объявлены правила. 

.title {
  color: white;
}

.title {
  color: orange;
}

Если специфичность одинаковая, обычно сработает правило, которое написано ниже.

Поэтому лучше держать стили организованными и не дублировать одинаковые селекторы без необходимости.

Position

Популярные значения свойства position
Значение Что делает
static Обычное положение элемента по умолчанию
relative Позволяет сдвигать элемент относительно его обычного места
absolute Позиционирует элемент относительно ближайшего positioned-родителя
fixed Фиксирует элемент относительно окна браузера
sticky Элемент ведёт себя как обычный, пока не «прилипнет» при прокрутке 
.badge {
  position: absolute;
  top: 12px;
  right: 12px;
}

Цвета и фоны 

.box {
  color: #ffffff;
  background: #1e293b;
}

.button {
  background: linear-gradient(135deg, #8b5cf6, #6d28d9);
}

В CSS цвет можно задавать по-разному:

  • red — имя цвета
  • #8b5cf6 — hex-цвет
  • rgb(139, 92, 246) — RGB
  • rgba(139, 92, 246, 0.2) — RGB с прозрачностью

Следи за контрастом текста и фона: текст должен хорошо читаться, а не просто красиво выглядеть.

Псевдоклассы и псевдоэлементы 

a:hover {
  color: tomato;
}

button:focus-visible {
  outline: 2px solid blue;
}

.card::before {
  content: "";
}
Частые псевдоклассы и псевдоэлементы
Запись Что означает
:hover Состояние при наведении
:focus / :focus-visible Состояние при фокусе с клавиатуры или другим способом
:first-child Первый дочерний элемент
::before Виртуальный элемент перед содержимым
::after Виртуальный элемент после содержимого

Мини-пример блока

Ниже пример простой карточки, где используются фон, отступы, скругление, тень и ограничение ширины. 

<div class="card">
  <h3 class="card-title">Заголовок</h3>
  <p class="card-text">Небольшой текст внутри блока.</p>
  <a class="card-link" href="#">Подробнее</a>
</div>
.card {
  max-width: 320px;
  padding: 20px;
  border-radius: 12px;
  background: white;
  box-shadow: 0 10px 30px rgba(0, 0, 0, 0.08);
}

.card-title {
  margin-bottom: 12px;
}

.card-text {
  margin-bottom: 16px;
}

Такие короткие готовые примеры помогают быстрее понять, как свойства работают вместе.

Частые ошибки новичков

Типичные ошибки в CSS
Ошибка Почему плохо Как лучше
Слишком длинные селекторы Стили сложнее поддерживать и переопределять Использовать понятные классы
Часто использовать !important Ломает предсказуемость каскада Разобраться со специфичностью и порядком правил
Задавать всё только в px Интерфейс хуже масштабируется Комбинировать px, rem, %
Убирать фокус у кнопок и ссылок Страница становится менее доступной Оставлять видимый :focus-visible
Красить текст без проверки контраста Текст может быть плохо читаем Проверять читаемость текста на фоне

Плохо

.header .menu ul li a {
  color: white;
}

Лучше

.menu-link {
  color: white;
}

Что важно запомнить

  • CSS отвечает за внешний вид элементов
  • Стиль строится из селектора, свойства и значения
  • Внешний CSS-файл — основной и самый удобный способ подключения
  • Классы обычно удобнее, чем длинные селекторы и id
  • margin — снаружи, padding — внутри
  • display, position и box model сильно влияют на layout
  • box-sizing: border-box делает размеры элементов удобнее
  • Важны каскад, специфичность и порядок правил
  • Не ломай доступность: оставляй видимый фокус и хороший контраст
